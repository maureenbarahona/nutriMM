# **App Name**: NutriSnap

## Core Features:

- Image Capture: Capture an image of food item.
- Nutritional Analysis: Use an AI tool to analyze the captured image against the INCAP database (https://www.sennutricion.org/media/tablas/INCAP.pdf) to determine nutritional information.
- Nutritional Value Display: Display the nutritional value in a clear and understandable format. Focus on key nutrients from the INCAP data.
- Manual Entry: Allow users to manually enter food items if image analysis fails.
- Food Item History: Store history of analyzed food items for user tracking. Enable storing custom quantities for more precise entries.

## Style Guidelines:

- Primary color: Soft Green (#A7D1AB) for health and nature association.
- Background color: Very light green (#F0FAF2), a desaturated variant of the primary.
- Accent color: Yellow-Orange (#F0AB60), brighter and more saturated than the primary for contrast and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif for both headlines and body text.
- Use clean and modern icons representing different nutrients and food categories.
- Clear and intuitive layout to present nutritional information effectively.
- Subtle animations for feedback and transitions, enhancing user experience.
